## ❤️爱心代码集合

### 🤩使用方法

#### 💌本地使用

对于 html 代码：将 **html** 代码文件下载至本地，并使用浏览器打开，

对于其他语言代码，使用对应的 IDE 运行即可得到效果。

#### 💌在线预览

- HTML 代码点击网址即可在线预览效果

注意：在`电脑端`显示效果最佳，手机端可能无法正常显示
